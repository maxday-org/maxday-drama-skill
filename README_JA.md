## 1つのコマンドで MaxDay を Agent ワークフローに導入

### AI Agent 経由でインストール（推奨）

以下のプロンプトを AI アシスタント（Codex CLI / App、Claude Code、Kimi Code/Claw、MiniMax Agent、Trae など）に直接送信すると、インストールと初期化を自動的に完了してくれます。

**プロンプト：**
> maxday-drama-skill をインストールしてください：`https://www.maxday.ai/maxday-drama-skill.zip`

### 手動インストール

ターミナルで実行：
```bash
npx skills add maxday-org/maxday-drama-skill
```
必要環境：Python 3.8+、追加の依存関係は不要。
初回利用時、Agent がメール認証コードによる MaxDay ログインをガイドします。

## すべての AI Agent で MaxDay を使いこなす

あなたの AI Agent はすでにアイデア出し、脚本作成、カット割り、プロンプト調整が得意です。しかし、実際の制作プラットフォームへの入口がまだありません。

maxday-drama-skill がその入口です。インストール後、Agent は「アドバイスを出す」や「プロンプトをコピーしてもらう」段階を超え、MaxDay で直接プロジェクトの作成、脚本のアップロード、アセット管理、画像・動画の一括生成、タスク進捗の追跡、生成結果のダウンロードができるようになります。

以前はアイデアを手動で MaxDay に持ち込んでいました。今は Agent が制作プロセス全体を MaxDay 上で実行します。あなたは**方向を示し、アクションを確認し、制作を続ける**だけです。

## 何ができるか

| やりたいこと | Agent が Skill を通じてできること |
|---|---|
| 短編ドラマを作る | プロジェクト作成、脚本アップロード、キャラクター/シーン抽出、一括画像・動画生成、完成品ダウンロード |
| キャラクター画像を一括生成 | モデル価格照会、画像生成タスクをループ送信、進捗追跡、一括ダウンロード |
| 動画を一括生成 | 参考画像/音声/動画をアップロード、動画生成タスクをループ送信、ステータス確認、ダウンロード |
| 脚本をアップロードしてアセット抽出 | 脚本ファイルアップロード、AI Agent 解析を起動、抽出タスクの状態を確認 |
| プロジェクトのアセットを管理 | フォルダ作成、画像/動画/音声を一括アップロード、リソース確認、削除と復元 |
| プロジェクト内で素早く画像・動画生成 | プロジェクト内でタスクを作成し、一括生成してダウンロード |
| 生成の進捗を追跡 | プロジェクト、タスク状態を確認、失敗タスクを特定 |
| コストと請求を確認 | 残高確認、AI 消費記録、取引履歴、WeChat チャージ |
| 組織とウォレットを管理 | 組織、メンバー、ウォレットサマリー、履歴、送金を確認 |

## このように使えます

以下は MaxDay にログイン後、Agent に直接送信できる完全なタスク例です。

### 1. 短編ドラマの一括制作

```
maxday-drama を使って、「シティクラッシュ」という短編ドラマプロジェクトを制作してください。脚本は ./script.docx にあります。

お願い：
1. MaxDay に新しいプロジェクトを作成、アスペクト比 9:16、スタイルは「映画的ライブアクション」；
2. 脚本をアップロードし、すべてのキャラクター、シーン、小道具アセットを抽出；
3. アセット抽出が完了したら、各主役に縦型のメインビジュアルを生成；
4. 最初の3シーンにそれぞれ5秒のオープニング動画を生成；
5. すべて完了したら ~/Documents/dramas/city-crush/ にダウンロード。
```

### 2. 画像の一括生成

```
maxday-drama を使って、画像を5枚生成してください。

要件：
- プロジェクト名「ネオンコンセプト画像プロジェクト」を使用/作成
- アスペクト比 9:16
- スタイル1：サイバーパンクネオンライティング
- スタイル2：ソフトナチュラルリアリズム
- 最も安い利用可能なモデルを使用
- プロンプト：サイバーパンク都市の雨の夜、濡れた街路にネオンが反射、映画的な構図、豊かなディテール
```

### 3. 単体動画の生成

```
maxday-drama を使って、動画を1本生成してください。

要件：
- プロジェクト名「ネオンショート動画プロジェクト」を使用/作成
- モデルは seedance2-pro-vvip
- アスペクト比 9:16
- 長さ 5 秒
- 解像度 480p
- プロンプト：サイバーパンク都市の雨の夜、濡れた街路にネオンが反射、カメラがゆっくり前進、映画的な光と影
```

### 4. キャラクター画像の一括生成

```
maxday-drama を使って、現在のプロジェクトの8人のキャラクターそれぞれに、異なるスタイルで2枚ずつポートレート画像を生成してください。

要件：
- 「ネオンキャラクターポートレート集」プロジェクトを使用/作成
- アスペクト比 9:16
- スタイル1：サイバーパンクネオンライティング
- スタイル2：ソフトナチュラルリアリズム
- 最も安い利用可能なモデルを使用
- 完了したらデスクトップの characters フォルダにダウンロード
```

### 5. 動画の一括生成

```
maxday-drama を使って、./storyboard/ ディレクトリにある10枚のストーリーボード画像から動画を一括生成してください。

お願い：
1. 「ストーリーボード動画集」プロジェクトを使用/作成；
2. すべての画像をそのプロジェクトの image フォルダにアップロード；
3. 各画像を開始フレームとして5秒の動画を生成；
4. プロンプトは統一して「cinematic camera movement, dramatic lighting」；
5. 完了したら ./output/videos/ に一括ダウンロード。
```

### 6. プロジェクト内での素早い画像生成

```
maxday-drama を使って、4枚のコンセプト画像を素早く生成してください。

お願い：
1. 「コンセプト画像探索集」プロジェクトを使用/作成；
2. そのプロジェクトで、以下のテーマのコンセプト画像を4枚生成：
   - サイバーパンクの街並み、雨の夜、ネオンの反射
   - 古風な中庭、月明かりの下で独り飲む
   - 宇宙ステーション内部、無重力で漂う宇宙飛行士
   - 海底の古代都市遺跡、水面を貫く光
3. 完了したらデスクトップにダウンロード。
```

### 7. コスト確認とチャージ

```
maxday-drama を使って、確認してください：
1. 現在のアカウント残高；
2. 最近の AI 消費記録をプロジェクト別に集計；
3. 残高が $2 未満なら、$10 の WeChat チャージ注文を作成。
```

### 8. 組織ウォレット管理

```
maxday-drama を使って、組織ウォレットを管理してください：
1. 組織一覧を表示；
2. メイン組織のウォレット残高と最近の取引を確認；
3. メンバー alice@example.com に100クレジットを送金。
```

## 安全に関する注意

クレジットの消費、データの削除、支払い注文の作成、送金、AI 生成のトリガーなど、すべての操作は実行前に確認されます。一括操作の意図を明確にした場合（例：「一括生成して」）、Agent は各ステップで確認を求めずに自動実行します。
